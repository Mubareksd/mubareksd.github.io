# Categories

Categories can store your bookmarks in custom made categories, which keeps your library neat and organized. You can name and sort them however you want according to your preference (ie. Genre, Reading Status, etc).

# Create a category

To make a category follow the steps below.

GUIDE

  1. Go to
    more_horiz
    More     →
    label
    Categories     .
  2. Press on + Add button on button right.
  3. Enter a name and press OK.

# Edit a category

To edit a category follow the steps below.

GUIDE

  1. Go to
    more_horiz
    More     →
    label
    Categories     .
  2. Long press a category.
  3. Tap
    edit
    EDIT     on the top-right.

# Change category order

You can change the order of Categories, such as bringing a certain Category to the top. This category will appear as the first tab on

collections_bookmark
Library .

To do this, follow the steps below.

GUIDE

  1. Go to
    more_horiz
    More     →
    label
    Categories     .
  2. Drag the category to your prefered position using the
    reorder
    Reorder     icon.

# Delete a category

To delete a category, follow the steps below.

GUIDE

  1. Go to
    more_horiz
    More     →
    label
    Categories     .
  2. Long press a category.
  3. Tap
    delete
    DELETE     on the top-right.

# Add manga to category

videocam
Click for video guide

To add manga to a category, follow the steps below.

GUIDE

  1. Long press the manga you want to add.
  2. Press the
    label
    Set categories     button.
  3. Select which category or categories you want it in and press OK.

    You can also add multiple manga to a category by selecting them when you see the

    label
    Set categories button.

# Remove manga from category

videocam
Click for video guide

To remove manga from a category, follow the steps below.

GUIDE

  1. Long press the manga that you want to remove.
  2. Press the
    label
    Set categories     button.
  3. Deselect the category or categories you want to remove it from and press OK.

    You can also remove multiple manga from a category by selecting them when you see the

    label
    Set categories button.
