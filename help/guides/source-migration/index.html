# Source migration

Migration is the process of moving manga between sources without losing progress. This is most often used when a source is no longer accessible or another source is more up-to-date.

Warning!

Always make sure to have a backup in case anything unexpected occurs.

# Migrating on Stable/Preview

Migrating from Manga

videocam
Click for video guide

  1. Tap into a Manga you would like to migrate.
  2. Go to
    more_vert
    Overflow     and tap Migrate .

    Tachiyomi will do a global search of all the sources you have installed and enabled.

  3. Select the Source that you'd like to migrate to by tapping the Manga thumbnail.
  4. Choose which data you want to transfer over, and you're done.

Migrating from Source

videocam
Click for video guide

  1. Tap into
    explore
    Browse     on the bottom navbar.
  2. Press the Migrate tab at the top next to Extensions .
  3. Select the Source that you'd like to migrate from.
  4. Tap the Manga you'd like to migrate from the Source.

    Tachiyomi will do a global search of all the sources you have installed and enabled.

  5. Select the Source you'd like to migrate to by tapping the Manga thumbnail.
  6. Choose which data you want to transfer over, and you're done.

Searching only pinned sources

You can restrict global searching to pinned sources if you wish to declutter your search.
To do this, go to

more_horiz
More →
settings
Settings →
explore
Browse then toggle the Only include pinned sources setting.

Pin sources in

explore
Browse using the
push_pin
pin icon next to the source name.

# Migrating on TachiyomiJ2K

Migrating multiple manga

videocam
Click for video guide

  1. Tap
    settings
    Settings →
    explore
    Sources → Source migration.
  2. Select the Source you'd like to migrate from and select All.
  3. Select the Sources that you'd like to migrate to and search by and tap the arrow at the bottom right.
  4. Choose which data you want to transfer over.
  5. Wait until all your Manga is found and hit the
    done_all
    at the top and you're done.

    If a manga is not found, or is wrong you can manually search it by pressing

    more_vert
    Overflow → Search manually.

Migrating a single manga

videocam
Click for video guide

  1. Tap into a Manga in your Library.
  2. Tap
    more_vert
    Overflow     → Migrate manga.
  3. Select the Sources you'd like to search and migrate to and hit the arrow at the bottom right.
  4. Wait until it is found and select
    done
    in the top right and you're done.

    If a manga is not found, or is wrong you can manually search it by pressing

    more_vert
    Overflow → Search manually
